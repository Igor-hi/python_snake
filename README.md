# Змейка (Snake Game)

Классическая консольная игра "Змейка", написанная на чистом Python без использования дополнительных библиотек.

### Особенности
- Полностью работает в терминале
- Автоматическое движение змейки (не нужно постоянно нажимать клавиши)
- Управление как стрелками, так и WASD
- Яркая графика с использованием Unicode-символов
- Подсчёт очков
- Проверка столкновений со стенами и собой

### Как запустить

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Скачайте или клонируйте репозиторий
3. Откройте терминал в папке с проектом
4. Запустите игру командой:

```bash
python "Путьк файлу"
